好的，我帮你整理一份 **VS Code 常用编辑快捷键清单（Windows & Mac）**，覆盖光标操作、行操作、复制/删除、选择、缩进、查找等，方便高效编程。

---

## **1️⃣ 光标移动**

| 操作          | Windows / Linux    | Mac               |
| ----------- | ------------------ | ----------------- |
| 光标左/右移动一个字符 | `←` / `→`          | `←` / `→`         |
| 光标上/下移动一行   | `↑` / `↓`          | `↑` / `↓`         |
| 光标移动到行首     | `Home`             | `Cmd + ←`         |
| 光标移动到行尾     | `End`              | `Cmd + →`         |
| 光标移动到文件开头   | `Ctrl + Home`      | `Cmd + ↑`         |
| 光标移动到文件末尾   | `Ctrl + End`       | `Cmd + ↓`         |
| 跳转到匹配括号     | `Ctrl + Shift + \` | `Cmd + Shift + \` |

---

## **2️⃣ 行操作**

| 操作         | Windows / Linux        | Mac                      |
| ---------- | ---------------------- | ------------------------ |
| 在当前行下方新建一行 | `Ctrl + Enter`         | `Cmd + Enter`            |
| 在当前行上方新建一行 | `Ctrl + Shift + Enter` | `Cmd + Shift + Enter`    |
| 删除当前行      | `Ctrl + Shift + K`     | `Cmd + Shift + K`        |
| 复制当前行      | `Shift + Alt + ↓ / ↑`  | `Shift + Option + ↓ / ↑` |
| 向上/下移动行    | `Alt + ↑ / ↓`          | `Option + ↑ / ↓`         |
| 注释/取消注释行   | `Ctrl + /`             | `Cmd + /`                |
| 块注释        | `Shift + Alt + A`      | `Shift + Option + A`     |

---

## **3️⃣ 选择操作**

| 操作       | Windows / Linux        | Mac                   |
| -------- | ---------------------- | --------------------- |
| 选中当前行    | `Ctrl + L`             | `Cmd + L`             |
| 选中所有相同单词 | `Ctrl + F2`            | `Cmd + Ctrl + G`      |
| 扩展选择     | `Shift + Alt + →`      | `Shift + Option + →`  |
| 缩小选择     | `Shift + Alt + ←`      | `Shift + Option + ←`  |
| 选中括号内内容  | `Ctrl + Shift + → / ←` | `Cmd + Shift + → / ←` |

---

## **4️⃣ 查找与替换**

| 操作    | Windows / Linux      | Mac                    |
| ----- | -------------------- | ---------------------- |
| 查找    | `Ctrl + F`           | `Cmd + F`              |
| 查找下一个 | `F3`                 | `Cmd + G`              |
| 查找上一个 | `Shift + F3`         | `Cmd + Shift + G`      |
| 替换    | `Ctrl + H`           | `Cmd + Option + F`     |
| 替换所有  | `Ctrl + Alt + Enter` | `Cmd + Option + Enter` |

---

## **5️⃣ 缩进与格式化**

| 操作          | Windows / Linux   | Mac                  |
| ----------- | ----------------- | -------------------- |
| 增加缩进        | `Ctrl + ]`        | `Cmd + ]`            |
| 减少缩进        | `Ctrl + [`        | `Cmd + [`            |
| 自动格式化文档     | `Shift + Alt + F` | `Shift + Option + F` |
| 触发建议 / 自动补全 | `Ctrl + Space`    | `Cmd + Space`        |

---

## **6️⃣ 多光标 & 多行编辑**

| 操作        | Windows / Linux    | Mac                |
| --------- | ------------------ | ------------------ |
| 在多处插入光标   | `Alt + 鼠标左键`       | `Option + 鼠标左键`    |
| 在下方添加光标   | `Ctrl + Alt + ↓`   | `Cmd + Option + ↓` |
| 在上方添加光标   | `Ctrl + Alt + ↑`   | `Cmd + Option + ↑` |
| 选择下一个相同单词 | `Ctrl + D`         | `Cmd + D`          |
| 选择所有相同单词  | `Ctrl + Shift + L` | `Cmd + Ctrl + G`   |

## **7️⃣ 其他常用操作**

| 操作        | Windows / Linux                 | Mac               |
| --------- | ------------------------------- | ----------------- |
| 打开命令面板    | `Ctrl + Shift + P`              | `Cmd + Shift + P` |
| 快速打开文件    | `Ctrl + P`                      | `Cmd + P`         |
| 切换终端      | `Ctrl + \`                      | `Cmd + \`         |
| 新建终端      | `Ctrl + Shift + '`              | `Cmd + Shift + '` |
| 关闭当前窗口/标签 | `Ctrl + W`                      | `Cmd + W`         |
| 重做        | `Ctrl + Y` / `Ctrl + Shift + Z` | `Cmd + Shift + Z` |
| 撤销        | `Ctrl + Z`                      | `Cmd + Z`         |
