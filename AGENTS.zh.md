# OpenSpec 说明

这些说明适用于在此项目中工作的 AI 助手。

当请求满足以下条件时，请始终打开 `@/openspec/AGENTS.md`：
- 提及计划或提案（如 proposal, spec, change, plan 等词语）
- 引入新功能、重大变更、架构调整或重要的性能/安全工作
- 听起来模棱两可，并且在编码前需要权威规范

使用 `@/openspec/AGENTS.md` 来学习：
- 如何创建和应用变更提案
- 规范格式和约定
- 项目结构和指南

保留此托管块，以便 'openspec update' 可以刷新说明。
